# Logo Images

Place your logo images here with transparent backgrounds and optimized sizes.

Required files:
- `farcaster.png` - Farcaster logo (transparent background, ~48x48px recommended)
- `base.png` - Base logo (transparent background, ~48x48px recommended)
- `eth.png` - Ethereum logo (transparent background, ~48x48px recommended)
- `btc.png` - Bitcoin logo (transparent background, ~48x48px recommended)

**Image Requirements:**
- Format: PNG with transparent background
- Size: 48x48px to 64x64px (optimized for web)
- File size: Keep under 50KB each for fast loading
